# Voitixa Tech PH

Static ecommerce website for a Philippines-focused tech store selling affordable gadgets, gaming accessories, smartphones, laptops, and tech accessories.

## Open the site

Open `index.html` in a browser.

No install step is required because the site uses plain HTML, CSS, and JavaScript.

## Buyer Accounts

Customers can create a new buyer account from the sign up form using a Gmail address and password. After sign up, the site opens the buyer dashboard automatically.
